# Blog System MySQL Database

A structured MySQL database designed for a blog system that stores users, articles, comments, authors, books, and categories in a clear and organised format. The database uses defined table relationships to support reliable storage and easy retrieval of content and user activity.

### Features

- Structured tables for users, authors, and books

- Tables for articles, comments, and categories

- Defined relationships between all main entities

- Supports organised data storage and fast data retrieval

- Designed using ER diagrams and a data dictionary

### Built With

- MySQL

- SQL

## Setup Instructions

### Requirements

- MySQL Server

- MySQL Workbench or any SQL client

git https://github.com/muskan-musa-khan/mysql-blog-system.git
